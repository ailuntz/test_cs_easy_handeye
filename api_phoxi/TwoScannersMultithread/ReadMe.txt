========================================================================
    CONSOLE APPLICATION : TwoScannersMultithread Project Overview
========================================================================

This is a simple example of how to find and connect more PhoXi Devices.

You will learn how to:

* check if PhoXi Control is running,
* obtain a list of devices connected to the network,
* connect to a devices attached to PhoXi Control if there is one
* capture scans from more threads.
